## O que o <b>AI IDE BAS</b> faz?
O AI IDE BAS ajuda a <b>gerar e validar diversos artefatos</b>. O trabalho é organizado por funções — cada função tem seus artefatos.

---

## Funções principais:
### <b>Analista de negócios</b>
- criar User Stories;
- criar Use Cases;
- criar diagrama de Atividade do processo de negócio em formato PlantUML;
- criar Acceptance Criteria;
- montar glossário do projeto;
- reunir informações de stakeholders do projeto;
- validar artefatos selecionados do analista de negócios.
### <b>Analista de sistemas</b>
- criar ERD (diagrama entidade-relacionamento);
- criar modelo de dados;
- criar diagrama de Sequência;
- criar especificação OpenAPI;
- criar especificação AsyncAPI para Kafka Message Broker;
- criar requisitos não funcionais;
- validar artefatos selecionados do analista de sistemas.
### <b>Arquiteto de soluções</b>
- criar diagrama de contexto;
- criar diagrama de contêineres;
- criar diagrama de componentes;
- validar artefatos selecionados do arquiteto de soluções.
### <b>Reviewer</b>:
- validar o projeto em qualidade de requisitos e artefatos de analista de negócios/sistemas;
- validar o projeto quanto a requisitos de cibersegurança;
- validar decisões arquiteturais;
- estimar custo da solução;
- validar o projeto sob a ótica de engenharia de suporte.

---

## Funções adicionais:
### <b>Helper</b>
- função de consultoria e dicas sobre o AI IDE BAS e o uso da extensão.
### <b>Designer</b>
- criar protótipos clicáveis.
### <b>Gerente de projeto</b>
- criar descrição textual do backlog do sprint;
- criar descrição de tarefa de desenvolvimento;
- diagrama de Gantt em formato PlantUML;
- diagrama de dependências em formato PlantUML.

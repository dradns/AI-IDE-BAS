## ماذا يمكن لـ AI IDE BAS أن يفعل؟

### التوليد:

- منطق الخلفية (Backend)
- مخططات ERD
- مخططات التسلسل
- مواصفات OpenAPI
- مواصفات AsyncAPI لـ Kafka Message Broker
- المتطلبات غير الوظيفية
- إنشاء قصص المستخدم (User Stories)
- حالات الاستخدام (Use Cases)
- مخطط نشاط للعملية بصيغة PlantUML
- أصحاب المصلحة في المشروع
- معايير القبول (Acceptance Criteria)
- معجم المشروع
- مخطط المكونات
- إطار مرجعي (Wireframe) (نموذج أولي، Mockup)

### التحقق:

- الأمن السيبراني (Cybersecurity)
- جودة المتطلبات ومخرجات محللي الأعمال والنظم (بواسطة محلل نظم أول)
- مراجعة القرارات المعمارية (Solution Architect)
- مراجعة من منظور مهندس الدعم (Support Engineer)

1. ثبّت الامتداد من VS Code Marketplace.
2. اربط مزود الذكاء الاصطناعي (متوافق مع OpenAI أو نموذج محلي).
3. ابدأ أول مهمة عبر لوحة الأوامر `AI IDE BAS: New Task`.

## الميزات الرئيسية

### أنماط متعددة

- Code Mode
- Architect Mode
- Ask Mode
- Debug Mode
- Custom Modes — أنشئ عددًا غير محدود من الشخصيات المخصصة.

### الأدوات

- قراءة/كتابة الملفات في مساحة العمل
- تنفيذ الأوامر في طرفية VS Code
- التحكم في المتصفح
- ربط الخدمات الخارجية عبر MCP (Model Context Protocol)

### الإعداد

- تعليمات وقوالب مخصصة
- نماذج محلية
- إعدادات التأكيد التلقائي للإجراءات

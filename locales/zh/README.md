## AI IDE BAS 能做什么？

### 生成：

- 后端逻辑
- ERD 图
- 时序图
- OpenAPI 规范
- Kafka Message Broker 的 AsyncAPI 规范
- 非功能性需求
- 用户故事（User Stories）
- 用例（Use Cases）
- PlantUML 格式的活动图
- 项目干系人
- 验收标准（Acceptance Criteria）
- 项目术语表
- 组件图
- 线框图（原型、mockup）

### 检查：

- 网络安全（Cybersecurity）
- 业务与系统分析师需求和产物质量审查（由资深系统分析师进行）
- 架构方案审查（Solution Architect）
- 支持工程师视角审查（Support Engineer）

## 快速开始

1. 从 VS Code Marketplace 安装扩展。
2. 连接 AI 提供商（OpenAI 兼容或本地模型）。
3. 在命令面板中运行 `AI IDE BAS: New Task` 开始首个任务。

---

## 关键能力

### 多种模式

- Code Mode
- Architect Mode
- Ask Mode
- Debug Mode
- Custom Modes — 创建不限数量的自定义“角色”。

### 工具

- 读取/写入工作区文件
- 在 VS Code 终端中执行命令
- 控制浏览器
- 通过 MCP（Model Context Protocol）连接第三方服务

### 配置

- 自定义指令与模板
- 本地模型
- 自动确认操作的设置
